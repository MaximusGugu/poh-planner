# POH Planner

A single-file, browser-based house floor plan designer with a live isometric 3D preview. Drag rooms onto a grid, lay out multiple floors, add doors/windows/arches, and view the result as a 3D model. All client-side, no build step or server required.

## Features

- **Drag-and-drop floor planner**: place rooms on a grid, rotate them with the mouse wheel while dragging, click to remove.
- **Multiple floors**: add and switch between floor tabs; move the whole lot as a unit.
- **Stairs**: connect floors with stair pieces.
- **House styles**: pick a material/style preset for the 3D model.
- **Isometric 3D viewer**: built with Three.js; orbit/zoom, toggle roofs, isolate a single floor.
- **Import/export layouts**: share or restore a layout via a text code.
- **Language toggle**: Portuguese (default) and English UI.

## Usage

Open `index.html` directly in a browser. No installation, dependencies, or build tooling needed. It loads Three.js from a CDN, so an internet connection is required.
